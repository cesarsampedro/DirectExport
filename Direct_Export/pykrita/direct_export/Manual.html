Direct Export Plugin for Krita

Overview

Direct Export is a Krita plugin that lets you export files to your last used location with all settings saved in your .kra file using a single shortcut.

Features

  • Quick export using a shortcut (customizable in Settings)
  • Remembers your last export location and settings per file stored within your .kra file
  • Supports multiple image formats (PNG, TGA, GIF, ...)
  • Visual feedback with temporary export confirmation
  • Optimized for game development workflow

Installation

To install the Direct Export plugin:

  1. Copy the folders actions and pykrita to your Krita resoruces folder
    • Windows: %APPDATA%\krita\
    • Linux: ~/.local/share/krita/
    • macOS: ~/Library/Application Support/Krita/
  2. Restart Krita
  3. Enable the plugin in Krita's Settings → Configure Krita → Python Plugins
  4. To customize the shortcut:
    • Go to Settings → Configure Krita → Keyboard Shortcuts
    • Search for "Direct Export" in the actions list
    • Click on the shortcut column and press your desired key combination
    • Click OK to save the changes

Usage

First Export

  1. Open the Direct Export docker (Settings → Dockers → Direct Export)
  2. Click the folder icon to select your export location and format
  3. Your export settings will be saved within the .kra file

Subsequent Exports

Once you've set up your initial export location, you can:

  • Press the shortcut key (or the arrow button at the docker) to quickly export to the same location
  • Use the folder button to change the export location & seetings whenever you want

Note: Export settings are saved per .kra file, allowing different export configurations for each of your source game assets.

Troubleshooting

Status Messages

  • If you see "-- Image not exported yet --" in the file path display, it means you need to set up the initial export location by clicking the folder icon.
  • If you see "-- Not a Krita file --", you need to save your document as a .kra file before using the plugin.
  • If the export button (arrow icon) is not visible, ensure you have set up a valid export path first.
  • If the file was exported with a newer version, the plugin will warn you, so install a more recent version.
